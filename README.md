# Expo Go Project with Google Sign-In

This project demonstrates integrating Google Sign-In in an Expo Go app using `expo-auth-session` and `@react-native-google-signin/google-signin`.

## Prerequisites

- Node.js (v16 or higher recommended)
- npm
- Expo CLI (`npm install -g expo-cli`)
- Expo Go app installed on your mobile device for testing

## Installation

1. Clone the repository:

```bash
git clone <your-repo-url>
cd <your-repo-folder>

npm install

if dependencies needed install this

npm install expo-auth-session
npm install @react-native-google-signin/google-signin
npx install expo-constants

then start

npx expo start
